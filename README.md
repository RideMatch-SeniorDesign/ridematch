# RideMatch

RideMatch is a dual-app rideshare platform built for our Engineering Senior Design project:
- **Rider mobile app** (find drivers, swipe to match, request rides)
- **Driver mobile app** (receive/accept ride requests, share location)
- **Admin web panel** (verify drivers, manage the platform)
- **Backend API + Database** (shared by all clients)

## Project Goals
- Give riders more control over who they ride with through **swipe-based matching**
- Support the full ride flow: match → request → accept/decline → track → complete → rate
- Include an **admin verification workflow** for driver activation

## Features (Target)
- Rider/Driver/Admin accounts & login
- Driver verification (admin approves driver)
- Swipe approve/reject for matching
- Ride request + driver accept/decline
- Real-time driver location during active ride
- In-app communication 
- Ratings (rider rates driver, driver rates rider)
- Payments
