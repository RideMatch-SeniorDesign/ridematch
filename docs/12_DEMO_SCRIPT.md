# Demo Script — RideMatch

## Goal
Deliver a clean 3–5 minute demo that shows the MVP spine working.

## Pre-Demo Setup
- Backend running and seeded (admin exists, at least 1 driver pending)
- Devices/emulators ready
- Stable Wi-Fi

## Demo Steps
1) Admin logs in
2) Admin approves a driver
3) Rider logs in and swipes/matches
4) Rider requests ride
5) Driver accepts
6) Rider sees driver location updating
7) Driver completes ride
8) Both rate each other

## Backup Plan (if something breaks)
- Use pre-seeded ride state
- Use emulator screenshots/video
- Use mocked location updates
