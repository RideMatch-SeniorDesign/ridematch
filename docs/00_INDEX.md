## Rules of the Road (how we use docs)
- **After every sponsor/advisor meeting:** add notes + action items to `03_MEETING_NOTES.md`
- **Anytime we choose a tool/approach/scope change:** log it in `02_DECISION_LOG.md`
- **Once per week:** update the “Current Snapshot” in `01_SEMESTER_TRACKER.md`
- **If a question affects architecture or scope:** put it in `11_QUESTIONS_TRACKER.md` before building
- **Every requirement we claim as “done” must have evidence:** link issue/PR/demo proof in `10_DELIVERABLES_REQUIREMENTS_TRACKER.md`
